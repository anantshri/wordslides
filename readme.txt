Create beautiful presentation right from your WordPress editor using markdown text and reveal.js magic

== Description ==
This plugin uses reveal.js as a backend service to allow users to create beautiful presentations using markdown as a input source. This allows us to leverage the flexibility of text and version controlling inbuilt in WordPress but also we can leverage awesome presentation magic using reveal.js backend.

== Installation ==
1. Download the zip file or search plugin in wordpress dashboard.
2. install and activate it.
3. a new post type is added in dashboard named as presentations.
4. create a new post and add your content using markdown format.
